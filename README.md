# hackathon アイデア

## 課題（やること）

このサイクルでAI（自分と似た存在）を最適化していく

第１段階　モチベUP

→楽しく(AIがToDo(スケジュール)、本、楽曲を提案)

第ｎ段階　習慣的に記録

→何かが育つ（ＵＩの変化（育成・コレクション要素））

→木などの植物（建物）の成長、バッチ（アチーブ）

最終段階　向上心UP

→習慣的に記録することでAIが分析（スコアリング）

→より良い自分になるために挑戦

---

## 機能要件

todo管理は二の次で（ベースはToDo登録）、一日のモチベーションをあげることがメイン

AIがTodoを提案する機能もあり

習慣的に記録することでAIが分析

カレンダーアプリを連携してデータを取得→AI分析からの提案

連続記録で成長する（木や建物）

記録が途切れたら、枯れるまでのカウントダウンがはじまって

その間に記録したら、もとの成長フェーズにもどる

カウントダウン切ったら、なくなる
